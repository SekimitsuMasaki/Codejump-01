## 🚀 セットアップ手順

このリポジトリをクローンして自分の環境で作業できるようにします。

### SSHでクローンする方法

#### （1）SSHキーの作成（まだの人だけ）
カリキュラム準備編-GitHub編を参考にSSHキーの作成

（2）SSHでクローン
```
git clone git@github.com:ユーザー名/リポジトリ名.git
```

（3）VSCodeで開く
